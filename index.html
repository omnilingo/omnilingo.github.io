OmniLingo

What is this?

OmniLingo is a project designed to help you get better at listening or writing in a language you are learning. It uses audio data from Mozilla's Common Voice project to create games that can help you practice your listening.

Get involved!

The project is completely free/open-source software and is developed on GitHub. If you like it, feel free to join us on #OmniLingo on Matrix or get the code and start hacking or file an issue on GitHub.
